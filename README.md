# Dashboard Financeiro

Projeto de dashboard financeiro desenvolvido com HTML, CSS e JavaScript puro.

## Funcionalidades

- Cadastro de receitas e despesas
- Cálculo automático de saldo
- Listagem de transações
- Remoção de transações
- Persistência de dados com localStorage
- Layout responsivo

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Objetivo

Este projeto foi desenvolvido para praticar fundamentos de front-end e lógica de programação, incluindo manipulação do DOM, eventos, funções, arrays, objetos e armazenamento local no navegador.

## Como executar

1. Baixe ou clone este repositório
2. Abra o projeto no VS Code
3. Execute com Live Server

## Aprendizados

- Estruturação semântica com HTML
- Estilização responsiva com CSS
- Manipulação dinâmica da interface com JavaScript
- Persistência de dados no navegador com localStorage
